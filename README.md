# ChatApp
ADS_Messenger

Massive Android Chat Messenger

Functionalities:
1) 1-1 Conversation
2) Group (up to 128 users)
3) Channel
4) Audio and Video Call
5) Status
6) Share media (audio, image, video, document)
7) Comment in channel (text and audio)
8) Notifications

Technologies used:
Java, Firebase, XML, Agora.io
